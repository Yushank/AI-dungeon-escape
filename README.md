# AI Dungeon Escape

A monorepo project built with Turborepo.

## Project Structure

```
ai-dungeon-escape/
├── game/                 # Turborepo root
│   ├── apps/
│   │   ├── backend/     # Backend application
│   │   └── frontend/    # Frontend application
│   └── package.json     # Turborepo configuration
└── README.md           # This file
```

## Development

```bash
cd game
npm install
npm run dev
```
